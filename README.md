## 📖 About

**Dua & Dhikr** is a beautiful mobile application that provides authentic Islamic supplications (Duas) and remembrances (Dhikr) from the Quran and authentic Hadith. The app offers a clean, user-friendly interface with powerful features to help Muslims stay connected with their faith.

## ✨ Features

### Core Features
- 📚 **500+ Authentic Duas** - Carefully verified from Hisnul Muslim, Sahih Bukhari, Sahih Muslim, and Quran
- 🔍 **Smart Search** - Search by name, Arabic text, or transliteration
- 🏷️ **Category Filter** - Browse by categories (Morning Adhkar, Evening Adhkar, Salah, Travel, etc.)
- ❤️ **Favorites** - Save your favorite duas for quick access
- 🌙 **Arabic Text** - Full Arabic script with beautiful Amiri font
- 📖 **Transliteration** - Easy pronunciation guide
- 🌐 **Dual Language** - English and Urdu translations

### Advanced Features
- 🔄 **Recent Searches** - Auto-saves last 5 searches
- 💡 **Search Suggestions** - Smart auto-complete suggestions
- 👁️ **View Counter** - Track popular duas
- 💾 **Offline Mode** - Works without internet
- 📱 **Responsive Design** - Works on all screen sizes
